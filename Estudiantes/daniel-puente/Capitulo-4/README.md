## Hilo conductor

El capítulo sigue el flujo real del servicio en el restaurante:

1. El usuario inicia sesión.
2. El camarero abre una mesa.
3. Se toma y edita la comanda.
4. Cocina trabaja desde el KDS.
5. El ticket se envía a caja.
6. Administración cobra el ticket.
7. La mesa se cierra y vuelve a estar disponible.
